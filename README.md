# Maya Drag-and-Drop Project Opener / Importer
### 📥 Скачать / Download

[[Скачать скрипт]](https://github.com/JollanXD/Drag-and-drop-Open-Import-For-maya-/blob/main/performFileDropAction.mel)



Скрипт добавляет удобное диалоговое окно при перетаскивании (Drag-and-Drop) файлов проектов `.ma` и `.mb` в окно Autodesk Maya. Вместо автоматического действия программа предложит вам выбрать: **Открыть (Open)** или **Импортировать (Import)** проект.

---

## 🛠 Installation Guide (EN)

1. **Close Maya:** Fully close the Autodesk Maya application.
2. **Navigate to the directory:** Go to the following folder:
   ```text
   C:\Program Files\Autodesk\Maya202X\scripts\others\
   ```
   *(Replace `202X` with your current Maya version, e.g., `2024` or `2025`)*.
3. **Backup the original file:** 
   * Locate the original `performFileDropAction.mel` file.
   * Rename it to `performFileDropAction.mel_BAK` for safety.
4. **Install the new script:** Copy the new modified `performFileDropAction.mel` file into this folder.
5. **Test it:** Launch Maya and drag-and-drop any `.ma` or `.mb` file into the viewport. A dialog box should now pop up asking you to **Open** or **Import**.

## 🛠 Инструкция по установке (RU)

Для установки скрипта вам понадобится заменить один из стандартных файлов программы. Пожалуйста, строго следуйте шагам ниже:

1. **Закройте программу:** Полностью выключите Maya перед началом установки.
2. **Перейдите в папку со скриптами:** Откройте проводник и перейдите по следующему пути:
   ```text
   C:\Program Files\Autodesk\Maya202X\scripts\others\
   ```
   *(Вместо `202X` подставьте вашу версию программы, например `2024` или `2025`)*.
3. **Создайте резервную копию (Бэкап):** 
   * Найдите в этой папке оригинальный файл `performFileDropAction.mel`.
   * Переименуйте его в `performFileDropAction.mel_BAK`. Это необходимо для безопасности, чтобы вы всегда могли вернуть всё назад.
4. **Установите новый файл:** Возьмите новый готовый файл `performFileDropAction.mel` и скопируйте его в эту папку.
5. **Проверьте работу:** Запустите Maya. Перетащите ваш `.ma` или `.mb` файл в окно программы. На экране должно появиться диалоговое окно с выбором **Open** / **Import**.
